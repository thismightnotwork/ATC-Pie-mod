Mouse gestures

Hover

  • strip: show comment in tool tip
  • radar scope: show selection-to-mouse info
  • unpinned navpoints and elements on the radar scope: show short name, and frequency/channel if navaid
  • entry in navigator or ATC list: show a tool tip with true heading and distance from radar centre

Click

Left button:

  • on strip, FPL, radar target or radar tag: select
  • on radar scope background: deselect
  • on radar navpoint:
    • plain click: fill navigator filter and instruction navpoint fields
    • with SHIFT if selection contains a strip with a recognised route: add/remove enroute waypoint
    • with ALT: send "proceed direct" instruction through CPDLC to selected callsign ("request direct" if teaching)
  • on text radio message:
    • own message: recall addressee and message in entry line
    • any ACFT message in teacher sessions: recall sender and message
    • other: recall addressee and focus message field
  • on "OK" button in instruction panel, with ALT: send instruction through CPDLC (or request if teaching)

Middle button:

  • on strip, FPL, radar target or radar tag: link to current selection if any and possible
  • with SHIFT on strip, FPL, radar target or radar tag (if selected and linked): unlink from rest of selection

Double-click

No modifiers:

  • on strip: open strip detail sheet
  • on radar target or tag: open linked strip if any
  • on rack, under strips: create new strip on rack
  • on loose strip bay background: create new strip in bay
  • on empty RWY box: start/reset separation timer
  • on rack column title: edit/delete rack
  • on radar navpoint: pin/unpin label
  • on navigator entry: indicate point on radar screen
  • on notification: notification-dependant
  • on ATC callsign: raise private text channel
  • on ATC phone line icon: switch outgoing status
    • if "idle": place call
    • if outgoing or call in progress: hang up (drop line)
    • if incoming: answer call
  • in status bar:
    • RDF label: show last detected signal info
    • METAR, wind or QNH label: raise weather dock
    • timer icon: open timers table

With SHIFT:

  • on strip: indicate linked ACFT or XPDR identification
  • on ATC callsign: indicate position if known
  • on radar target: remove hdg/alt./FL/speed assignments
  • on radar tag: reset text box position
  • on radar scope background: add custom text label (annotate radar)
  • on custom text label: remove label

With ALT:

  • on strip, radar target or radar tag: open current/last CPDLC window for callsign

Click & drag

Left button:

  • strip:
    • within/between racks, loose bays or runway boxes: rerack/reorder/reposition flight strip
    • drag loose strip with SHIFT: move and keep within bay, pushing edges as needed
    • drop on shelf: shelve strip, e.g. releasing a contact with no handover
    • drop on ATC callsign, without ALT: send strip to ATC
    • drop on ATC callsign, with ALT and active data link: data authority transfer or handover instruction (as teacher: request handover)
  • radar tag:
    • without SHIFT: move linked strip (see above)
    • with SHIFT: move tag
  • radar target: vector/instruct (*)
    • without SHIFT: assign heading or instruct taxi if detected on ground
    • with SHIFT, dragging vertically: assign alt./FL
    • with SHIFT, dragging horizontally: assign speed
  • loose strip bay or radar scope background (unless locked): pan view

Right button:

  • on radar scope
    • without SHIFT: quick heading and distance measuring tool + show ground elevation if available
    • with SHIFT as teacher: create new traffic
  • in tower view window when not following an aircraft (FlightGear input): look around

(*) Notes on click & drag vectoring:

  • with ALT (except taxi): send instruction through CPDLC (or request if teaching)
  • press right button while dragging to abort
  • in solo sessions if voice recognition turned off: action instructs callsign
  • as teacher: all actions take direct control of aircraft (no strip callsign look-up)
