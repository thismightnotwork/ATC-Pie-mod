Keyboard and text input

Action keys and shortcuts

Many shortcuts correspond to menu actions, likely referenced directly in the application menus. Others can be found in the list and tables below.

  • Ctrl: radio PTT key. Use in solo sessions for voice instructions when recognition is turned on, and in teacher/student sessions when integrated radio support is activated by teacher. In FlightGear and FSD sessions: acts on all radios with "kbd PTT" ticked.
  • Alt: CPDLC key. Combines with mouse actions like vectoring, strip handover and instruction panel buttons (see "Mouse gestures" tab), as well as the ENTER key to open the current/latest connection window for selected callsign.
  • Single dot (".") in an airport input field: shortcut for base airfield. Fills the input with the current ICAO position (AD mode only).
  • Pause: Works in solo, teaching and playback sessions (pausable clock). FG and FSD sessions are tied to real time; student sessions tied to the teacher's clock.

Point specification

In various occasions, e.g. to position your CTR radar centre or draw radar background images, you will want to specify location points. To specify a point, you may either:

  • write lat/lon Earth coordinates separated by a comma with no space, e.g. 35.8765,-90.567 or 48°51'24''N,2°21'03''E (you may replace the °/'/'' symbols by d/m/s respectively to avoid encoding problems);
  • spell a navpoint code (VOR, airfield...);
  • prefix a navpoint code with a parenthesed type, e.g. "(VOR)ABC", to disambiguate between navpoints with identical names, selecting by type (accepted types: AD, VOR, NDB, FIX, ILS, RNAV);
  • append an ICAO region filter to a navpoint code after "@", e.g. "ABC@LK";
  • use the "namedpoint~refpoint" syntax to choose between navpoints with identical names, selecting the closest to a reference point;
  • append a ">radial,distance" suffix to any base point specification to move it by a distance in NM along a true radial ("~" has precedence over ">").

For example:

  • 48°51'24''N,2°21'03''E is the centre of Paris;
  • SULED>180,1.5 is a point 1.5 NM due true South of navpoint SULED;
  • IA~KIAD is the point named IA that is nearest to KIAD (with a 200 NM map range from KPIT, you see two points "IA" so the name alone is ambiguous);
  • (NDB)MEN~(AD)LFMT is the NDB whose code is MEN that is the closest to the airfield whose code is LFMT.
