Text aliases

Predefined aliases

Session-dependant:

  • $ad Long name of the base airfield (AD mode only).
  • $atis Last recorded ATIS letter.
  • $decl Local magnetic declination.
  • $elev Base airfield elevation in feet (AD mode only).
  • $frq Current publicised frequency.
  • $helipads List of active helicopter pads (AD mode only).
  • $icao Current location code (AD or CTR).
  • $me Your radio callsign (location setting), or text callsign if none set.
  • $metar Last METAR string.
  • $qfe Current ground pressure at airfield in hPa (AD mode only).
  • $qnh Current sea level pressure in hPa.
  • $qnhg Current sea level pressure in inches of mercury.
  • $runways Summary of DEP/LDG use of runways (AD mode only).
  • $rwyarr List of runways selected for arrivals (AD mode only).
  • $rwydep List of runways selected for departures (AD mode only).
  • $ta Transition altitude.
  • $tl Lowest flight level, given current QNH and TA.
  • $utc Current universal coordinated time in hhmmZ format.
  • $vis Reading of METAR prevailing visibility.
  • $wind Reading of METAR wind.

Selection-dependant:

  • $cruise Cruise altitude/FL written on the strip or linked FPL.
  • $dest Destination written on the strip or linked FPL.
  • $dist Distance between radar contact and airfield (AD mode only).
  • $nseq Sequence number on rack.
  • $qdm Bearing to airfield from radar contact (AD mode only).
  • $rack Name of rack on which strip is sequenced.
  • $route Contents of the strip or linked FPL route field.
  • $rwy Runway for which strip is boxed (AD mode only).
  • $sq Assigned transponder code.
  • $valt Vectored altitude or FL.
  • $vhdg Vectored heading.
  • $vspd Vectored speed.
  • $wpnext Next waypoint on route.
  • $wpsid First enroute waypoint if "SID" appears in first leg spec.
  • $wpstar Last enroute waypoint if "STAR" appears in last leg spec.

Custom aliases

If not in the list of predefined text aliases above, any "$alias" word in a text message will trigger a search for a line starting with "alias=" in the following texts, in order:

  • general notes;
  • local notes;
  • selected strip comments.

The remainder of the first line found is substituted in place of the alias. This allows to define custom text aliases on all possible levels: user/general, location (AD or CTR), and selection. If the substitution is unsuccessful, ATC-pie will open an edit box so that you can review your message before sending it.
